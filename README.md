# Tugas JavaScript Interaktif + GitHub Challenge

Project ini dibuat untuk memenuhi challenge Git dan GitHub.

## Fitur
- Tombol untuk mengubah tema (light/dark mode)
- Tombol untuk memberikan sapaan ke pengguna
- Manipulasi DOM sederhana dengan JavaScript

## Teknologi yang digunakan
- HTML
- CSS
- JavaScript
- Git & GitHub

## Instruksi Menjalankan
Cukup buka file `index.html` di browser.

## Refleksi Singkat
Bagian tersulit bagi saya adalah memahami hubungan antara HTML dan JavaScript,
terutama saat memakai event listener dan manipulasi DOM.

Setelah mencoba, saya mulai paham alurnya.
Momen paling menyenangkan adalah ketika tombol berhasil mengubah tema
dan sapaan muncul sesuai nama yang dimasukkan.
